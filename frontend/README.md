# Blackcoffer Data Visualization Dashboard

## Tech Stack
- Backend: Node.js, Express, MongoDB
- Frontend: React (Vite), Tailwind CSS
- Charts: Chart.js
- Deployment: Render + Vercel

## Features
- KPI metrics (Intensity, Likelihood, Relevance)
- Topic, Region, End-Year filters
- Interactive Bar & Bubble charts
- PESTLE analysis
- Drill-down insights table
- MongoDB-backed API
